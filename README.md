### Only HTML website

Create an HTML only website with at least two pages about yourself or a topic that you are familiar with.

Properly use all of the following tags to create structured and meaningful content.

- `<html>`
- `<head>`
- `<body>`
- `<p>`
- `<strong>`
- `<em>`
- `<img>`
- `<a>`
- `<div>`
- `<ul>`
- `<ol>`
  - `<li>`
- `<table>`
  - `<tr>`
    - `<th>`
    - `<td>`
